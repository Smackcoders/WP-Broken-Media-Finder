# Scope: Broken Media Finder

## Plugin Name

Broken Media Finder

## Plugin Purpose

Broken Media Finder helps WordPress admins, agencies, bloggers, and WooCommerce store owners detect missing images, broken attachment links, and unused media files.

The plugin should provide a lightweight maintenance tool that scans site content, reports broken media references, helps replace missing images with a placeholder, and exports scan results for cleanup work.

## Target Users

- WordPress maintenance teams
- Agencies
- Bloggers
- WooCommerce stores
- Website admins
- Content managers
- SEO teams
- Migration teams
- Client support teams

## MVP Scope

The free community version should include:

- Scan posts and pages for missing images
- Detect broken attachment URLs inside post content
- Detect missing featured images
- List unused media files
- Replace missing image with placeholder
- Export scan report as CSV
- Admin dashboard summary
- Manual scan button
- Scan history
- Basic settings page
- Security with nonce, capability checks, sanitization, escaping
- Translation-ready strings
- Basic PHPUnit/WP test coverage

## Future Pro Scope

- Auto-repair moved media
- Cloud media support
- WooCommerce product image scan
- Scheduled weekly scan
- External image validation
- CDN media validation
- Bulk media replacement
- AI-assisted media cleanup recommendations
- Xapplets automation integration
